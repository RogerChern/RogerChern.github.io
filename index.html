Linear Regression

发表于 2015-08-19   |   分类于 ML

Symbols

  • $X$ — the design matrix
  • $\theta$ — the parameters of model
  • $n$ — number of features size(X,1)
  • $m$ — number of training samples size(X,2)
  • $x^{(i)}$ — the i-th training sample X(i,:)
  • $x_j$ — the j-th feature X(:,j)

Hypothesis

$$h_\theta(x)=\theta^{T}x=\theta_0x_0+\theta_1x_1+\cdots+\theta_nx_n$$

Cost function (Sqaured Error)

$$J(\theta)=\dfrac{1}{2m}\sum_{i=1}^m[h_\theta(x^{(i)})-y^{(i)}]^2$$

Gradient Descent

Normalization

$$x_i:=\dfrac{x_i-\mu_i}{\sigma_i}$$

Batch version

$$\theta_j:=\theta_j-\alpha\dfrac{1}{m}\sum_{i=1}^m(h_\theta(x^{(i)})-y^{(i)})x_j^{(i)}$$
